jekyll安装和使用
-----------
##安装
```
gem install jekyll
```
##基本使用
```
~ $ gem install jekyll
~ $ jekyll new myblog
~ $ cd myblog
~/myblog $ jekyll serve
# => Now browse to http://localhost:4000
```
